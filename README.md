### 安装
1. 下载 Webstorm-2016-10-23.jar
2. 在 WebStorm 中，选择 File -> Import Settings
3. 打开 WebStorm -> Preferences -> Plugins -> Browse Repositories，搜索 Material Theme UI，安装
4. 重启 WebStorm，完成安装
